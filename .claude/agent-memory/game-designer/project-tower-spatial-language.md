Tower_EarthCoast_01 空间语言规范（V8，PM 批准 2026-06-10）：

- **三秒判读测试**：手电扫一遍 3 秒内必须能识别空间类型和出口方向。白盒不达标改几何，不堆道具。
- 走廊 4m 宽（PM 决定），身份靠压顶截面（2.2–2.4m）+ 电缆桥架 + 柱节奏，不靠贴图。
- 六个光锚：值班台暖橙 / P-01 红灯 / B 梯钠灯频闪 / 塌角天光 / 样板岛冷白 / 生态柱青绿。
- 样板岛冷光 `#E8F0F4` 是全图唯一精装区，材质断层是叙事锚。

**Why:** 第一人称手电主照明，玩家读截面而非平面图。几何语法是信息的第一载体。  
**How to apply:** [[project-style-lock]] 所有资产提案都先验证截面几何是否满足空间身份，不能靠贴图补救几何。
