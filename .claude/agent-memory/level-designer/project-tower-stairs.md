PM 反馈"从门里进来不是直接看到楼梯，很怪"——根因是第一跑与入口门轴线错位。

**Why:** A 梯草案第一跑在西半（x26..28），但 D10 门洞中心在 x=28；玩家进门需左转才看到踏步。B 梯草案同理（且平台仅 1.2m 宽，坡度 35°）。

**How to apply:** 楼梯设计铁律——第一跑底端必须出现在入口门正对方向轴线 1m 以内，无需玩家转身超过 30°。

关键坐标决策（已写入 `design/levels/tower-stair-redesign-v2.md`）：
- A 梯：第一跑改为中线 x27..29（与 D10 门洞轴对齐），U 形折返，25° 坡，顶平台 z28..29.5 紧贴 D35。
- B 梯：受 x 方向 4m 宽限制，坡度下限为 31°（合规）；平台改为西侧全宽 2m、z 向 6m；入口平台扩宽至 4m（z18..22）。

井道通高 5.6m 的净空问题：顶平台贴出口门，玩家踏上即脱离封闭顶板约束，净空条件满足。
