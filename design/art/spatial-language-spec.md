## 1. 核心原则

游戏是第一人称、手电筒主照明、黑暗环境。玩家看不到平面图——他们读的是
**截面**：宽度、天花高度、顶部轮廓、门洞比例、光的种类。因此：

1. **几何语法先于道具语言**——空间身份必须先由截面几何传达，道具只做加强，
   不做补救。
2. **走廊保持 4m 宽（PM 决定）**——宽度不分级；身份靠"压顶"与节奏：
   走廊的感知高度必须明显低于房间。
3. **每个空间只有一个功能身份**（functionClass）：
   `死端房 / 枢纽 / 走廊 / 路口 / 楼梯核 / 中庭塌角 / 脚手桥 / 毛坯板`。
   枢纽（穿行空间）零战利品；战利品只放死端房。
4. **三秒判读测试**（验收标准）：玩家用手电扫一遍周围 3 秒内必须能答出
   "我在哪类空间、出口大概在哪个方向"。白盒阶段不达标 = 改几何，不是堆道具。

## 3. 门洞语法

- **门宽 ≤ 所在墙面净宽 50%**（4m 墙 → 最大 2.0m）。超过 50% 时空间边界在
  感知上消失，两侧读作同一空间——这是 V7"房间里出现走廊状区域"的直接原因之一。
- **穿行枢纽禁止同轴对穿**：两个关键门不得在平行对墙上中心间距 < 2m
  （V8 数据校验 V8-C2 强制）。
- 门洞分级：枢纽/主廊交接 = 2.8m 无门扇洞口；普通房间 = 2.0m 粗糙混凝土门洞
  （无门套）；走廊⇄走廊合流（J）= 柱间收窄的全宽贯通，无门框。
- 每个房间门口设**阈值标记**：门框横梁 + 地面材质分界线（在门外即可见）。

## 5. 致命公司方法借鉴（仅方法，不复制内容）

1. 截面变化是身份第一信号（走廊管线压顶 vs 房间净顶）——零新资产成本。
2. 门洞比例分级制造入口等级感。
3. 光的分布图案即空间身份：走廊=线性节奏；房间=单点聚焦；大厅=高位下射大阴影。
4. 面包屑：每个关键转向节点放一个不可移动的特征残留物。
5. 恐惧节奏=体积韵律：窄→宽切换（B 梯封闭 4m → F2 毛坯板 44m 是全图最强的一次，保留并放大）。
