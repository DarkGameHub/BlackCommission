# Game Concept — Black Commission (Black Outsourcing / Outsourcing Incident Unit)

> **Index file.** This points to the canonical design docs so framework skills
> can find the concept. Do not duplicate content here — edit the source docs.

## One-liner

A 1–4 player co-op "commission-running" game: a nearly bankrupt office takes
increasingly absurd outsourced jobs to stay afloat. Identity: **Municipal Debt Noir**.

## Signature Loop

`HQ office → office computer (accept job) → gear/shop → board dispatch van →
in-van transit → mission site → objective / partial-return choice → van return →
HQ settlement`
