# Black Commission Style Lock v2 — 低保真工业恐怖（Lo-Fi Industrial Horror）

**状态**: LOCKED — PM Yan Dai 拍板（2026-06-10）
**取代**: `black-commission-style-lock-v1.md`（半写实方向，废止）
**回归**: `retro_industrial_horror_style_guide.md`（2026-05-27 PM 原始方向）的低保真路线
**与 art-bible 的关系**: `design/art/art-bible.md` 仍是执行圣经（色板纪律、灯光语法、
判读测试、命名规范全部有效），但其中**保真度条款按本文档修正**（见 §6）。

## 1. 一句话定位

低模强剪影的合作工业恐怖：致命公司所属的"低保真合作恐怖"品类的形体语言，
装着 Black Commission 自己的市政欠债灵魂（Municipal Debt Noir 色板 + 烂尾/公文/讨债题材）。

## 2. 风格坐标

**是**：低多边形几何、可见纹素的低分辨率贴图、平面/简单材质、强光池+暗角的可读黑暗、
廉价公共机构的旧与脏。

**不是**：
- 半写实 PBR（v1 路线，废止）——禁止 1K 微细节、扫描级表面
- 极端 PS1 像素恐怖（不做顶点抖动、不做仿 CRT 全屏滤镜）
- 卡通/高饱和（REPO 的比例夸张和明亮对比不进来）

**红线（继承 AGENTS.md，约束力不变）**：Lethal Company 仅作品类与生产方法参照。
凡是 LC 截图里能看见的——飞船、怪物、终端、UI、道具、地图——一律不得复制。
我们共享的是"低保真合作恐怖"这个品类语言，不是它的资产。

## 3. 几何规则（继承 retro 指南，数值照旧）

- 模块化套件建图；强剪影优先，先形体后细节
- 小道具 20–300 三角面；门/柜/桌 100–800；人形 700–2500；怪物 1000–4000（剪影承担主要工作）
- 重要玩法物体 = 强剪影 + 唯一一个点缀色
- 在玩法尺度和构图成立之前，禁止写实微细节

## 4. 贴图与材质规则（本次定案的核心）

| 项 | 规则 |
|---|---|
| 分辨率 | 大面（墙/地/顶）**≤256px**，道具 ≤256px，可见纹素颗粒是特征不是缺陷 |
| 平铺 | 世界尺度锚定，约 **2m 一循环**（白盒由 builder 的 MPB 平铺实现） |
| 通道 | Albedo 为主；**不用 normal/AO/metalness 微细节贴图**（金属感用颜色+smoothness 表达） |
| 粗糙度 | 高粗糙（smoothness ≤0.3），无光泽塑料/镜面 |
| 平色 | 允许——但暴露表面必须带脏旧（颜色变体/贴图颗粒/角部积尘），干净平色只准出现在"新换的/精装的"叙事表面（如样板岛） |
| 来源 | 项目内 ambientCG 贴图降采样使用；AI 生成贴图同样压到 ≤256 |

## 5. 色板与灯光（全部继承 art-bible，零修改）

- 混凝土灰 60% / 军绿 20% / 旧木 15% / 锈 5%
- 主点缀 = 暖钨丝橙黄（`#FFAB40`–`#FF8C00`）；CRT 绿 `#6CFF5F` 只准电子屏；
  印章红 `#C23A2B` 只准纸面/标识；危险临边 = 陈旧安全黄 `#C8A020`–`#A87E10`
- 灯光只有三种：5000K 冷白 / 3000K 钨丝暖 / CRT 绿；单手持光源判读测试照旧
- 烂尾楼空间语言（`design/art/spatial-language-spec.md`）继续有效

## 6. 对 art-bible 的修正条款

以下 art-bible 条款按本文档读：
1. 风格定位测试 §"style position test" 中 "not low-poly" → 改为 "**is** lo-fi low-poly"
2. 材质测试 "20–40% weathering 写实风化" → 改为"脏旧通过低分辨率颗粒+颜色变体表达"
3. 贴图分辨率分级表 → 数值改读为**上限 256**（原表作为远期高清重制参考保留）
4. "Prohibited: clean flat color on exposed surfaces" → 保留，但满足方式按 §4 平色行

## 7. 验收测试（替代 v1 的四项）

1. **品类测试**：截一帧给陌生人看，应答出"低保真合作恐怖游戏"而不是"写实游戏的草稿"
2. **身份测试**：同一帧里能找到至少一个 BC 身份元素（市政色板/公文/欠债标语/钨丝橙黄）
3. **单光源测试**（继承 art-bible）：黑暗中单手电能认出资产是什么
4. **纹素测试**：凑近 0.5m 能看见贴图颗粒；退到 4m 表面读作整体色块
