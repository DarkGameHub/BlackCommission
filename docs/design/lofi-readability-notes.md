## 1. 黑轮廓（Outline Post-Process）对合作恐怖可读性的影响

### 必须有轮廓的对象类型

| 类型 | 理由 |
|---|---|
| **可交互物**（开关/柜门/拾取品） | 手电筒扫过时轮廓是"可以摸"的唯一瞬时信号；低保真贴图没有高光细节做替代 |
| **队友轮廓**（全黑暗中） | 合作恐怖的"友军辨认"是生死优先级，4 人挤同一走廊时轮廓比姓名浮标更直觉 |
| **威胁/怪物**（近距离）| 低多边形轮廓明确，但暗环境下手电边缘依然会丢失形体；轮廓做二次保险 |

### 不该有轮廓的对象类型

| 类型 | 理由 |
|---|---|
| **背景建筑几何**（墙/柱/梁） | 走廊截面本身就是可读信号（见 spatial-language-spec §1）；加描边会把每块砌体都变成"可交互暗示"，制造假信息 |
| **道具/场景杂物**（料堆/电缆/防尘布） | 它们只是导航障碍和叙事密度，不需要行动提示 |
| **光锚装置**（灯具本体） | 灯自己会发光，轮廓多余；且轮廓在光晕附近会形成难看的黑边与光晕叠加 |

### 全屏统一描边 vs 选择性描边

**结论：选择性描边（按标签）。**

- 全屏统一描边在低保真几何上产生"黑色漫画分镜"效果，与 BC 目标色调（工业恐怖 / Municipal Debt Noir）不符；对比 REPO 那种故意夸张的轮廓风，这不是我们的品类定位。
- 选择性描边：可交互物 = 常驻细轮廓（0.5–1px world space）；队友 = 透视穿墙轮廓（颜色锁定玩家色）；威胁 = 进入 8m 阈值时激活，退出消失。
- 成本：需要 URP Renderer Feature 按层分两遍描边，实现复杂度 M 级，但维护成本远低于"到处都描边后再一个个撤"。

## 2. 手电筒主照明 + 低保真贴图 + 黑暗下三秒判读防劣化规则

背景：spatial-language-spec §1.4 要求玩家手电扫 3 秒内识别空间类型与出口方向。256px 贴图颗粒 + 暗环境 + 移动光源会让以下情况劣化：
- 贴图颗粒在手电边缘衰减区产生"糊掉"的灰色噪声，空间截面边界消失。
- 低分辨率 Albedo 在小角度掠射光下几乎一片平色，丢失走廊/房间的层次差异。

**防劣化规则（3 条）**

**规则 A — 轮廓用几何，不用纹理**
空间身份（"我在走廊还是房间"）的第一信号必须来自截面几何变化（电缆桥架下压、柱节奏、门洞比例），不依赖贴图细节。256px 贴图在手电运动中随时失读，但 2.2m 压顶 vs 2.8m 净顶的截面差在任何照明条件下都成立。

**规则 B — 光锚色相必须高于周围墙面饱和度至少 30 个百分点**
`#E8A33D` 钨丝暖橙 vs 混凝土灰 `~#8A8A7A`：饱和度差 ~45%，手电扫到边缘时光锚依然可识。若未来引入更多光锚，色相距离 ≥ 60° 且饱和度差 ≥ 30%，否则合并或删除光锚（宁少勿混）。

**规则 C — 可交互物必须有形体剪影差，不能只靠轮廓描边**
手电筒移动时描边 post-process 会因深度采样频率而闪烁。可交互物的"第一眼可识"需来自：高于周围几何的剪影（开关突出墙面、柜子高于地面）或唯一点缀色（style-lock §3："重要玩法物体 = 强剪影 + 唯一一个点缀色"）。描边是第二层保险，不是第一层。

## 3. 纹理提升优先级排序

预算有限时，按玩家注视时间和可读性收益排序。

### 值得花预算（高注视 + 高收益）

| 表面类型 | 理由 |
|---|---|
| **可交互物表面**（柜门正面、开关面板、拾取道具） | 玩家走近操作时停留最久；256px 够用但脏旧变体必须清晰（防"糊烂"误读为不可交互） |
| **地面**（走廊和关键房间） | 第一人称视角下地面占屏比高；导航时眼睛沿地面找边界；走廊混凝土振捣纹应清晰可读 |
| **门洞阈值标记**（门框横梁 + 地面材质分界线） | spatial-language-spec §3 要求门外即可见；这条线是判读测试最直接的锚点 |

### 糊弄就行（低注视 + 低收益）

| 表面类型 | 理由 |
|---|---|
| **顶棚**（非特征区域） | 第一人称几乎不仰视；走廊压顶的桥架剪影已经承担身份功能，顶棚贴图贡献接近零 |
| **大面积背景墙**（非叙事墙） | 这里的 256px 颗粒感是风格特征，不是问题；提升只会让它看起来"写实草稿"（style-lock §2 明确禁止） |
| **外墙幕墙龙骨**（毛坯板 Open 区）| 玩家在这区域快速穿越；极稀疏道具+料堆掩体才是注意力重心 |

## 4. 与 LC 拉开差异的最低成本抓手

当前已有：市政色板 / 钨丝橙黄点缀 / 公文题材。但"低保真+若加黑轮廓"的品类同质化风险是真实的——手电扫过混凝土走廊，第一眼和 LC 依然像。

### 最低成本的三个差异化锚点

**锚点 1 — 文字密度（纸面 / 标识 / 印章）**
BC 的题材是"公文/讨债/委托书"。走廊墙面放烂尾工程标牌、欠款催缴贴纸、施工许可到期印章（印章红 `#C23A2B`）。这些是纸面叙事，成本极低（贴图+mesh decal），但能让玩家第一眼认出这不是废弃工厂而是"被讨债机构接手的烂尾楼"。LC 的场景是工业宇宙飞船，这个语义域完全不重叠。

**锚点 2 — 钨丝橙黄的光池形状**
LC 的主光源是冷白荧光灯管（线型）。BC 的钨丝橙黄（`#FFAB40`–`#FF8C00`）光池是点状/球形，天花高处悬挂，产生圆形热点 + 向外快速衰减的黑暗边界。这个"暖点 + 急剧暗角"的图案在第一帧就区别于 LC 的线型冷白。

**锚点 3 — 样板岛冷光 `#E8F0F4` 作为全图叙事异物**
F2 销售中心洁净地砖 + 完好吊顶 + 嵌入灯具是全图唯一"精装"区域，和毛坯走廊形成强烈材质断层。玩家走到 F2 时立刻感受到"人类曾经用力装修过这里"，这是 BC 特有的"郊区楼盘烂尾"叙事时刻。LC 没有这种材质反差，它的全图保持统一工业肮脏度。

### 执行优先级
1. 先落实文字贴图（decal 系统或 UV 直贴）——成本最低，差异化效果最强。
2. 确保钨丝橙黄光池形状（点光源，不换成线型荧光）。
3. 样板岛区域保持精装材质，不要"为了统一风格"把它也搞成毛坯。
